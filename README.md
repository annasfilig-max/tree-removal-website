# Tree Removal Template

Static, deployable website template. Plain HTML + Tailwind CDN + vanilla JS — no build step.

## Use
1. Open `index.html` in a browser, or upload the folder to any host.
2. Find-and-replace bracketed placeholders like `[Your Company Name]`, `[Phone Number]`, `[Email]`, `[City, State ZIP]` with real details.
3. Swap placeholder images in `assets/images/` — see `assets/images/README.txt` for slot list.
4. Replace form action in every `<form>` with your Formspree endpoint.
